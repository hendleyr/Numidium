oculus-rest

What is oculus-rest?

oculus-rest is a third-party http server that provides head tracking data between the Oculus Rift™ and your PC. It is what allows our scene to detect when you are moving your head in order to move the camera accordingly.

To download or find out more, check out the oculus-rest repository on GitHub.

Download

Download oculus-rest here: github.com/possan/oculus-rest.

Locate Files

Navigate to the directory containing the oculus-rest project.

Run

Run the provided executable for your operating system.

...But that didn't work!

The GitHub repository also includes project files for Visual Studio(Windows) or XCode(OSX) so that you can build and run the files locally.
